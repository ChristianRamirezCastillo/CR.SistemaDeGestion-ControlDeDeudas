SISTEMA DE GESTION
Control de Deudas

Version 1.0
-----------
Sistema para controlar las deudas que tienen los clientes en los bancos


Datos del Sistema
-----------------------
Plataforma:	Escritorio	
Lenguaje:		Visual Basic.Net
Bse Datos:	SQL Server 2008 R2
IDE:		Visual Studio 2013 Update 4
Sistema Opetativo:	Windows 8.1 Pro


Contenido del Sistema
------------------------------
El sistema contempla 4 modulos

- Módulo de Cartera
- Módulo de Clientes
- Módulo de Usuario
- Módulo de Empleado
- Módulo de Reporte - Cliente


Cargar Base de Datos
-----------------------------

Opcion 01:
- En la carpeta DATA se encuentra el backup (.bak) para restaurarlo al SQL Server 2008 R2

Opcion 02:
- En la carpeta DATA se encuentra la base de datos (.mdf) para restaurarlo al SQL Server 2008 R2

Opcion 03:
- En la carpeta DATA se encuentra los querys (.sql) para ejecutarlos en al base de datos
- Respetar el orden para ir ejecutandolo (terminado el .sql 3, podra ejecutar los demas que no tienen numeros)
- Debera importar la data desde un excel que se encuentra en la carpeta EXCEL, respetando los numeros de cargas, es decir
  * cargar primero cliente, seguido de empresa y por ultimo producto para evitar errores
